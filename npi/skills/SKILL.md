# Skill: FIO Output Parser

This skill parses the JSON output generated by the Flexible I/O Tester (FIO) and extracts key performance metrics.

## Purpose
To reduce the size of FIO output data and make it easier for agents or analysis scripts to understand performance results without parsing the full, verbose JSON structure.

## Usage

### Python
```python
from skills.fio_parser import parse_fio_output

results = parse_fio_output("path/to/fio_output.json")
for result in results:
    print(f"Job: {result['job_name']}, BW: {result['bw_mibps']} MiB/s")
```

### Command Line
```bash
python3 skills/fio_parser.py path/to/fio_output.json --output text
```

## Input
- A file path to a JSON file generated by FIO with `--output-format=json`.

## Output
A list of dictionaries (or JSON array), each containing:
- `job_name`: Name of the FIO job.
- `operation`: Type of operation (`read`, `write`, etc.).
- `bw_mibps`: Bandwidth in MiB/s.
- `iops`: I/O operations per second.
- `mean_lat_ms`: Mean latency in milliseconds.
- `p99_lat_ms`: 99th percentile latency in milliseconds.
